JOGO DA MEMÓRIA

O código JavaScript desenvolve um jogo da memória interativo. Foram utilizadas listas para armazenar as imagens das cartas e variáveis de controle para o estado do jogo (como pontuação e bloqueio de cliques). As funções manipulam o DOM para criar as cartas, embaralhá-las, lidar com cliques e verificar pares. Também foi usado setTimeout para dar tempo ao jogador de visualizar as cartas antes de escondê-las novamente. O jogo é iniciado automaticamente ao carregar a página, integrando HTML, CSS e JavaScript para uma experiência dinâmica. Data: 08/06
