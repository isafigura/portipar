Validador de Senha Forte

Foi criado um formulário simples para o usuário digitar uma senha. Ao enviar, o JavaScript utiliza expressões regulares (RegExp) para verificar se a senha atende aos critérios de segurança: mínimo de 8 caracteres, presença de letra maiúscula, letra minúscula, número e caractere especial. Cada critério é testado e, em seguida, uma lista de mensagens é exibida, indicando com símbolos visuais quais foram atendidos ou não. O DOM é manipulado para atualizar essas mensagens dinamicamente, proporcionando um feedback claro e imediato ao usuário. Data: 11/06
