Editor de texto

Insira o conteúdo que deseja editar no campo a seguir.

Editor de texto simples desenvolvido com HTML, CSS e JavaScript. Permite aplicar formatações como negrito, itálico, sublinhado, centralização e listas. O objetivo é demonstrar conhecimentos adquiridos em sala de aula, utilizando recursos como document.execCommand, uma função do JavaScript que permite executar comandos de edição diretamente no conteúdo selecionado pelo usuário. Com ela, é possível modificar textos em áreas editáveis da página, simulando o comportamento de editores como Word ou Google Docs.
