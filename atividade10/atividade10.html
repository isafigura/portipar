Sorteador de Nomes

Este código define uma função sortearNome(), que realiza o sorteio a partir dos nomes inseridos no input. Ao ser chamada, a função recupera o valor digitado no campo com o ID "entradaNomes". Em seguida, essa string é dividida em uma lista de nomes usando o método .split(","), e cada nome é "limpo" com .trim() para remover espaços em branco antes e depois. Também é aplicado um filtro para eliminar entradas vazias. Depois disso, o código verifica se há pelo menos um nome válido na lista. Se não houver, uma mensagem de aviso é exibida na tela, pedindo que o usuário insira pelo menos um nome. Se a lista estiver correta, a função escolhe um nome aleatório usando Math.random() e Math.floor() para gerar um índice válido dentro do array. Por fim, o nome sorteado é exibido no elemento com o ID "resultado".
