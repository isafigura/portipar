Controle de Visibilidade (toggle)

Foi utilizado JavaScript para manipular as classes CSS das seções, ativando e desativando a classe .show com o método classList.toggle(). Cada botão possui um atributo data-target que indica qual seção será afetada. No CSS, as transições foram implementadas utilizando as propriedades transition combinadas com max-height e opacity, garantindo que o conteúdo apareça e desapareça de forma fluida. O uso de max-height com um valor fixo alto permite a animação da expansão e recolhimento do conteúdo e o opacity adiciona uma mudança visual mais agradável. Assim, o projeto une manipulação do DOM, eventos de clique e transições CSS.
